# Streak Tracker

A habit tracking app that lets you compete with friends. Built with React, Supabase, and OneSignal.

## Current Status

Under active development

## Tech Stack

- React + Vite
- Supabase (database + auth)
- OneSignal (notifications)
- Netlify (hosting)

- [ ] Daily streak tracking
- [ ] Friend comparisons
